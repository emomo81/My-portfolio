# **App Name**: MomoSphere Portfolio

## Core Features:

- Interactive 3D Hero Scene: Showcase a primary 3D interactive scene powered by Spline, featuring dynamic elements and acting as the main visual anchor of the hero section.
- Dynamic Project Gallery: Present projects in an interactive grid with individual project cards displaying images, titles, tags, and links, enhanced with hover effects and animations. Projects are easily managed from a data file.
- Animated Experience Timeline: Display work experience and education in a scroll-triggered, vertically animated timeline for an engaging narrative flow.
- Customizable Skills Showcase: Categorize and display technical and soft skills through interactive cards or tags with hover effects and staggered entrance animations.
- Smooth Preloader & Navigation: Implement a full-screen preloader with an animated reveal, and a sticky navigation bar with active section highlighting and smooth scrolling.
- Scroll-Triggered Cinematic Animations: Utilize Framer Motion and Intersection Observer to implement subtle parallax, fade-in, slide-in, and staggered entrance animations across all content sections, creating a cinematic feel.
- Responsive Contact Form: Provide a clean, glass-morphism styled contact form for user inquiries, alongside social media links, designed to be fully responsive.

## Style Guidelines:

- Overall Theme: Dark mode with a primary background of deep charcoal (#0a0a0f, HSL: 225, 29%, 3%).
- Primary Accent Color: A vibrant electric blue (#3b82f6, HSL: 217, 91%, 60%) for interactive elements, highlights, and calls to action, providing high contrast against the dark background.
- Secondary Accent Color: A contrasting cyan (#06b6d4, HSL: 191, 91%, 43%) as an analogous companion, offering visual depth and alternative accentuation.
- Headlines and prominent text will use 'Space Grotesk' (sans-serif) for a modern, techy, and bold presence. Body text and longer descriptions will use 'Inter' (sans-serif) for clear readability and a clean, objective feel.
- Clean, modern icons from 'Lucide React' or 'React Icons' for clarity and visual appeal, integrated with hover animations for micro-interactions.
- Generous whitespace and precise element alignment to foster a premium, organized aesthetic. Implement 'glass-morphism' for card backgrounds and navigation blur effects, enhancing depth. Utilize parallax for background elements to create immersive visual experiences, particularly in the hero and 3D sections.
- Extensive use of Framer Motion for cinematic scroll-triggered entrance animations, hover effects (e.g., lift, glow, tilt), micro-interactions (e.g., custom cursor, magnetic buttons), and smooth page transitions. Implement smooth loader with an animated logo reveal and progress indication. Integrate seamless 3D motion via Spline components with appropriate fallbacks and lazy loading.
